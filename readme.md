# Flask test

This is a demo project for flask

创建虚拟环境，在虚拟环境操作

```bash
python -m venv venv
```

Run the init-db command:

```bash
flask init-db
```

> Initialized the database.

Windows设置FLASK_APP环境变量

```bash
set FLASK_APP=flaskr
```

Linux设置FLASK_APP环境变量

```bash
export FLASK_APP=flaskr
```

启动app

```bash
flask run
```

开启 debug

```bash
export FLASK_ENV=development
```
